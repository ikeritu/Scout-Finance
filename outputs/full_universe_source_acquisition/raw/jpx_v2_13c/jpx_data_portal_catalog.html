JPxData Portal

JPxData Portal provides a comprehensive catalog of all paid and free data provided by JPX Group companies and partner companies. It also features articles on how to utilize each dataset and announcements regarding new services that have been launched.
You can find the data you need by using the keyword search or filtering the list via the “Data Catalog” menu at the top of the JPxData Portal screen.
In addition to data, you can also search for listed stocks and disclosure information.
We hope this portal will help you discover new data related to the Japanese market and listed stocks you may not have known about before.

Please visit the JPxData Portal website here.

Main features

Features Overview
Data Catalog
  • Search over 200 types of data using simple keywords such as "Stock Prices," "Derivatives," "Margin Trading," and "ESG."
  • View a comprehensive list of data update frequency, historical data coverage period, file formats (PDF, CSV, Excel, etc.), and pricing.
  • Sample data and usage guides are available for some datasets.
Use Cases
  • Articles are provided, including "How to Use Data," which introduces analysis examples using various datasets, and "Similar Data," which explains differences between related datasets such as stock price data and issue master file information.
Companies
  • View basic information on individual stocks, disclosure information, and trends in earnings and stock prices.
  • In addition to company names and codes, you can search for stocks using keywords like "Cloud" and "DX" powered by generative AI technology.
  • Download the latest list of listed securities in CSV format.
Disclosure (J-LENS Series)
  • Search TDnet disclosure information the past 1 to 5 years (depending on the plan).
  • Perform natural language searches using AI on TDnet disclosures from the past 3 years.
  • For earnings reports and other disclosures provided in HTML format, browser machine translation can be easily used. Articles are also available on how to use browser machine translation and important usage notes.
Related Sites
  • Useful websites related to the securities market, primarily those provided by JPX Group and other organizations, are introduced, with plans for continuous expansion.
