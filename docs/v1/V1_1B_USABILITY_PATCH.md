# v1.1B — Usability Patch

## Objective

Improve the usability of the Quickstart after the v1.1A real local use trial.

## Scope

Documentation-only patch.

## Changes

- Adds an initial freeze validation step.
- Separates generic usage from ticker examples.
- Adds the generic command format for manual decisions.
- Adds allowed manual statuses.
- Adds commands to open `manual_review_summary.md` and `final_review_pack.md`.
- Adds a list of important files.
- Makes the no-financial-advice warning more visible.

## Explicit non-goals

- No scoring changes.
- No filter changes.
- No pipeline recalculation.
- No OpenAI execution.
- No external API calls.
- No yfinance calls.
- No release freeze changes.
