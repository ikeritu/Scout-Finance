- [ ] Ejecutar `check_phase4g_stability.py`.
- [ ] Confirmar que `app.py` compila.
- [ ] Confirmar JSON en `outputs/analyses`.
- [ ] Confirmar que Comparar empresas carga tarjetas.
- [ ] Confirmar que Histórico muestra ticker.
- [ ] Confirmar que Ajustes muestra Sistema listo.

## Señales de prudencia

- [ ] Confianza baja.
- [ ] Datos insuficientes.
- [ ] Evidencia débil.
- [ ] Muchas limitaciones.
- [ ] Riesgo creciente.
- [ ] Categoría cambia a peor.

## Nunca interpretar como

- [ ] Recomendación de compra.
- [ ] Recomendación de venta.
- [ ] Asesoramiento financiero.
- [ ] Sistema automático de inversión.
